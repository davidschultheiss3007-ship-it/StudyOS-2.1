# Quellenablage

Dieser Ordner ist fuer fachliche Originalquellen gedacht, z. B. PDF-Skripte,
Folien oder Aufgabenblaetter, aus denen StudyOS-Inhalte erstellt werden.

## Struktur

```text
sources/
  semester-4/
    asset-management/
    behavioral-finance/
    business-english/
    human-resources/
```

## Benennung

- Modulordner entsprechen den Modul-Slugs unter `src/modules/`.
- Dateinamen klein und eindeutig halten, z. B. `skript-01-grundlagen.pdf`.
- Wenn sinnvoll Semester, Stand oder Datum im Dateinamen ergaenzen.
- Originalquellen nicht mit App-Assets aus `public/` oder `src/` vermischen.
